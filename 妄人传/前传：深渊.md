# 尾声.


“爸爸，林幽阿姨死了，你当时其实是非常开心的，对吧？”离离歪着头，问道。

“开心，也不开心。我当时还没认识你边叔叔，根本没有有效清洗和保存尸体的手段，结果，你林幽阿姨就像我日记里写的一样，腐烂生虫，最后没法用了。不然，你现在还能看到一身女王装扮的她，她会永远年轻。”赵一琅轻轻捋着自己的白胡须。

“那，那些你拍的照片和视频……”

“那些艺术品一直都在哟，不过，你现在太小了，还不能体会其中的美感和奥妙，等你过完成年礼吧。”

“嗯，还有，后来你杀死了好几个穿着靴子的女孩子，还和她们生宝宝，警察不管你吗？”

“管啊，起源觉醒后，爸爸快活了不到半年，就被捉拿归案啦。爸爸只是个书呆子，没有一身功夫，没有接受过反侦察训练，也没有你边狩叔叔手眼通天的关系网和高科技支撑，就靠本能，那怎么行呢。”

“我看日记上说，后来的新闻报道，‘变态杀人犯赵一琅被判处死刑，并于某年某月某日执行枪决’，爸爸，赵一琅不就是你真正的名字吗？”

“那是你边叔叔托关系找的替死鬼，杀人犯必须枪决，以平民意。况且，这样一来，我就永远欠你边叔叔一条命啦，以后他要我做什么事，我都得做。”

赵一琅苦笑着，用右手摸着右眼窝的眼罩，又看向自己已经变为义肢的双腿和左臂。

“边狩叔叔真是神通广大呀，我去过边叔叔和李叔叔的生物实验室，感觉里面好厉害。爸爸，你怎么和边叔叔认识的？”

“哈哈，那是另一段故事了，改天再讲给你听。至于那个实验室，等你长大以后，只要缘分到了，你自然会经常去那里玩的。”

“嗯，好想快点长大呀，我要变得比爸爸和边叔叔更强，早日遇见属于我的林幽阿姨，然后狠狠地和她的尸体谈恋爱。”

“好呀，只要爸爸还有一口气，就会陪着你长大的。对了，离离，你知道我给你取的大名有什么含义吗？”

“含义？我只知道我叫赵震离……听起来还挺霸气的……”

“震卦为雷，有崩裂之相却无崩裂之实，震苏苏，震遂泥，震往来；离卦为火，流连互灼，兴于同极而枯于旦夕，突如其来，终为日昃之离。震离相合，则为丰时，星火燎原，雷电交加，万物勃勃生发，必能成就大业……”

“爸爸，我困了……”

“啊，是爸爸话太多了，你去玩儿吧。记得八点之前回来。”

望着赵震离蹦蹦跳跳的背影远去，赵一琅颤巍巍地从怀中取出一本古旧的纸质书，放在夕阳下，细细端详着。

“说起来，这本书，到现在我还没有读完呢。还差哪一章来着？对了，‘卡列宁的微笑’。”

书的封面印着醒目的标题和作者：不能承受的生命之轻。米兰·昆德拉。

翻到扉页，上有一行手写的，娟秀的小字：

生日赠与挚爱知己，林幽。

再翻到目录页，上面有两行他自己所写的，从书中摘录的句子：

只能经历一次的生命，就等于从未活过。

在这个不存在轮回的世界上，一切都预先被谅解了，一切也就被卑鄙地许可了。
