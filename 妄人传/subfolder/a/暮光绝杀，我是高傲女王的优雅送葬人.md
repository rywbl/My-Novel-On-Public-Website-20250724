
# 34. 跨海大桥的挽歌

“就在这儿吧。”

雨燕熄灭了引擎，透过车窗，看着下面那片深不见底的黑色海洋，满意地点了点头，“干净，壮阔，不会有人来打扰。”

“怎么，改变主意了，让我就这样死在半路吗？”红夫人看了一眼深不见底的江面，自嘲地笑道，“确实，这里比烂在泥里干净多了。水流会带我入海……”

眼前的场景陷入了一种奇异的静谧，只有肖邦的夜曲还在不知疲倦地流淌。

“不动手吗？”红夫人转过头，那双曾经精于算计的眼睛此刻清澈得如同初生，“还是说，你想听我求饶？”

“不。”雨燕的声音依旧冷淡，像是在陈述一件公事，“我在等你的烟抽完。”

红夫人愣了一下，随即低笑出声。她看了一眼指间那支只剩下三分之一的女士细烟，深吸了一口，吐出一个完美的烟圈。

“你变了，雨燕。以前你是把好刀，但也仅仅是把刀。现在的你……更像是个诗人。”

“死人总是有很多时间去思考。”

“是啊……时间。”红夫人感叹着，眼神有些恍惚，“我这一辈子都在抢时间，抢钱，抢男人，抢地狱的门票。到头来，还是两手空空。”

最后一口烟雾散尽，她将烟蒂轻轻丢弃在疾风骤雨里，仍由那黯淡光点消失在那黑色夜空，她的动作轻柔得像是在掐灭一段过往。

她裹紧了身上的银狐披肩，踩着那双红底高跟鞋，一步步走向大桥的边缘。暴雨将她的身体淋得通透，那昂贵的丝绒礼服紧紧贴在身上，勾勒出她依旧曼妙的身姿，却也让她看起来像是一朵在风雨中瑟瑟发抖的残花。

雨燕不知何时撑起了那把黑伞，走下车，站在了她的身后。

黑伞遮住了头顶的暴雨，为红夫人撑起了一方最后的小小天地。

“谢谢。”红夫人没有回头，她扶着湿滑的栏杆，俯瞰着脚下的万丈深渊，“没想到最后给我送终的，居然是被我害死的人。”

“这是交易的一部分。”雨燕冷冷地回答，“你给了我情报，我给你体面。”

“体面……”红夫人咀嚼着这个词，突然转过身，背靠着栏杆，直视着雨燕那双藏在墨镜后的眼睛。

“那我们就把这出戏演到底。”

她抬起手，解开了颈间那条价值连城的红宝石项链。

那是她权力的象征，是她无数次出卖灵魂换来的战利品。

“拿着。”她将项链递到雨燕面前，“这也是交易的一部分。别让那些臭男人把这东西拿去换酒喝。”

雨燕没有拒绝，伸出戴着黑色皮手套的手接过项链，随手揣进了风衣口袋。

“还有这个。”

红夫人又解开了身上的银狐披肩。

那是她最后的温暖。

随着披肩滑落，她那洁白的双肩和后背完全暴露在冰冷的雨水中。她打了个寒颤，却倔强地挺直了脊背，像一只濒死的天鹅。

“这件衣服……是我第一次见那个男人时穿的。”她低头看着身上那件已经被雨水浸透成暗红色的礼服，“那是二十年前的事了。那时候我还是个想当歌星的傻女孩，以为爱情能拯救一切。”

她抬起头，雨水顺着她精致的妆容滑落，分不清是雨还是泪。

“动手吧，雨燕。用你最擅长的方式。”

红夫人闭上了眼睛，微微仰起头，露出了修长白皙的脖颈。

那姿态，既像是献祭，又像是引颈受戮。

雨燕看着她，那双死水般的眸子里终于泛起了一丝微澜。

她没有拔枪。

对于这样一个骄傲了一辈子的女人，枪决太过粗暴，也不够“体面”。

雨燕缓缓抬起左手，黑色的皮手套在雨水中泛着幽光。

她的手掌没有掐向脖颈，而是顺着红夫人湿透的腰线滑落，精准而轻柔地拍向了那红裙之下的大腿外侧——那是红夫人藏匿最后底牌的隐秘之处，也是她防身的剧毒所在。

只是轻轻一巴掌，拍向了红夫人红裙之下的湛蓝药剂，让那裙底之下的药剂，轻轻的浸润红夫人那高贵之下的隐秘之处。

“啪。”

一声极轻的脆响。

她上前一步，身体几乎贴上了红夫人，姿态亲密得如同情人间的耳语，但手上的动作却带着绝对的死亡意志。

黑色的雨伞遮蔽了天空，将两人笼罩在阴影之中。

“再见，夫人。”

雨燕的声音低沉如耳语。

没有痛苦的挣扎，没有血腥的喷溅。致命的处决在无声无息之间就已经完成，夺命仇敌的性命，就这样以一种种极其精准、极其温柔的角度，奇妙而精巧的完成最终的复仇。

雨燕的手掌隔着湿冷的红裙，精准地击碎了缝合在裙底内衬中的那支湛蓝药剂。

只有一声极轻的液体包裹被挤压破碎的水滴之声，被掩盖在雷声与海浪声中。

随着裙底药剂的碎裂，冰蓝色的液体瞬间浸润了红裙的布料，紧紧贴上了红夫人温热的肌肤。那种极致的冰寒顺着毛孔瞬间钻入，与她的肌肤产生了剧烈的化学反应。

# 35. 致命的优雅 

雨水不是落下来的，是砸下来的。

跨海大桥的钢索在狂风中震颤，发出低沉如鲸鸣的嗡响。整座桥仿佛一头被缚于深渊之上的巨兽，在暴风雨中喘息、挣扎，却始终不肯屈膝。桥面两侧的景观灯早已熄灭，唯有远处城市天际线的霓虹在雨幕中晕染成一片模糊的金红——那是人间的灯火，温暖、喧嚣、与她们无关。

海风像是被某种看不见的利刃割裂，跨海大桥上的霓虹灯影在这一刻显得格外刺眼。红夫人并没有因为刚才那一瞬的致命接触而失态。

她甚至微微抬了眼，感受着身底那股致命的寒意，从大腿根部逐步蔓延。

红夫人她站在观景台最边缘，左手扶着冰冷的铸铁栏杆，指节因用力而泛出青白。身上的红裙早已湿透，紧贴身体，勾勒出瘦削却未垮塌的轮廓。水珠顺着锁骨滑进领口，在胸前汇成细流，又沿着腰线蜿蜒而下。

“真是好手段。”红夫人轻轻放下了手，嘴角的笑意反而深了几分，那是一种看透了生死赌局的从容，“不用针孔，不用气雾，现在的年轻人，杀人的技艺倒是越来越艺术了。”

“为了配合您的身份，粗鲁的手段自然不合适。”雨燕的声音依旧平静得像是一潭死水。

她能清楚的感觉到裙底那股尖锐的凉意，沿着神经束开始向上攀爬。但也能感受到那里的布料吸饱了水后的沉重，那是毒剂已经彻底融入了身底的隐秘的症结，一切像一层冰冷的铅衣裹在身上，让每一次呼吸都带着滞涩的阻力。

她缓缓抬起右手，不是去扶栏杆，而是理了理被雨水打湿的鬓发。动作从容，指尖掠过耳后，将一缕乱发别至耳后。

她知道这是什么，毕竟这一切是她亲手塑造的配方，“暮光”。

“你倒是有心。”红夫人的声音很轻，几乎被雨声吞没，却异常清晰，“连毒……都选我熟悉的。”

雨燕站在三步之外，黑伞垂落，隔开一片干燥的阴影。她没靠近，也没说话，只是静静看着红夫人用拇指抹去下眼睑被雨水晕开的一道灰痕——动作轻，却带着一种近乎偏执的整理欲。

“熟悉的东西，才不会让你失态。”雨燕终于开口，声音穿透雨声，清晰如刀，“你教过我：真正的死亡，不该有狼狈的余韵。”

红夫人沉默。雨水顺着她的发梢滴落，在肩头洇开一小片深色。她闭了闭眼，再睁开时，瞳孔深处已无波澜。


# 36. 她乡归途

“车在那边。”雨燕说，“该上路了，时不我待，想必你也知道。”

裙底的寒意开始扩散，那是毒剂开始生效，逐步扩散的征兆，
毒素已然入体，在她的血管里蜿蜒爬行。

红夫人抬眼，唇角扯出一丝极淡的弧度，她松开栏杆，站直。双腿微颤，但被长裙遮掩得严严实实。她迈出第一步，高跟踩碎一地水光。

雨燕没再问，转身带路。伞始终微微倾向红夫人一侧——不是体贴，而是不能让她以滴水之姿踏入自己的座驾。

路基湿滑，红夫人的脚踝开始麻木，小腿肌肉不受控地抽搐。她咬住内唇，每一步都像踩在刀尖，用疼痛维持清醒。眼下她不能摔倒。哪怕死，也要站着死。

雨燕就在后方两步处默默等她，洪夫人没有回头，她倒也不催促。她可以清晰的听见对方身前那细微的、压抑到极致的喘息，但也能感知到对方每一步前行的迟疑与坚持。只不过眼下，她们都没有任何共青，只有一种近乎冷酷的欣赏。

似乎这才是她记忆中的红夫人——那个在血泊中还能补妆的夫人。

“你知道吗？”红夫人忽然开口，声音因寒冷而略显沙哑，“之所以当初选定的是你，不是因为你够狠，是因为你够静。”

雨燕脚步未停。“静，才能足够狠。”雨燕终于加快了脚步，微微侧过半张脸，让那雨水顺着她的下颌线滑落。

两人继续前行。风更大了，卷着咸腥的海水味扑面而来。红夫人的视线开始模糊，毒素深入肌肤，无可救药。她看到雨燕的背影在雨中晃动，像一幅被水浸湿的水墨画。

只不过，眼下，她们也走过了最后的几步路，座驾，咫尺之遥。

她侧过身，黑色的风衣衣摆在风中微微扬起，腕表轻触门把手。车锁应声开启，车内顶灯柔和亮起。

副驾侧车门被雨燕拉开，沉重的机械声在空旷的桥面上回荡。

“谢……谢……”

红夫人停在车门前，深深吸了一口气，她没有犹豫，左手扶住车顶框稳住身形，右腿先入，身体顺势滑进车内，尽管动作因虚弱而略显迟滞，但仍然刻意避开了任何踉跄或倚靠。

车内暖意扑面而来，却没能驱散她骨髓里的寒。她感到毒素正向心脏逼近，每一次心跳都变得沉重而遥远。

空气中弥漫着一股淡淡的皮革味和陈年的雪茄香气，这是红夫人最熟悉的味道，此刻却像是某种祭奠。

# 37. 最后的交易

红夫人微微欠身，提起裙摆，以一种无可挑剔的姿态滑入后座。雨燕随即关上车门，绕过车尾回到了驾驶室。

她收伞时手腕一抖，积水尽数甩落，伞面瞬间半干。坐进车内，皮革发出轻微的摩擦声。

雨燕撑着黑伞走近，伞面隔绝了大部分风雨，投下一片阴影。

“毒发的感觉如何，夫人？”

她的声音毫无感情，深沉的像似一块冷静的冰，传递给她的眼神就像是在观察实验品。

红夫人深深地看了一眼雨燕。这个年轻人的眼神里没有胜利者的狂喜，也没有杀手的冷酷，只有一种令人心悸的职业化精准。

她理了理那条暗红色的丝绒披肩，动作优雅得仿佛刚才被宣告死亡的并不是她，缓缓的将湿透的银狐披肩从肩头褪下，叠放在腿上，遮住礼服下摆的水痕。

“安全带。”她没看红夫人，只伸手按下中控屏上的“除雾+脚部暖风”。暖流无声涌出，驱散寒气，却不至于让湿衣蒸腾出狼狈的雾气。

红夫人没应声，只是抬起手，缓慢而准确地将安全带扣入锁扣。金属咬合的“咔哒”声，在寂静车厢里格外清脆——像为这场雨夜对峙，落下第一枚休止符。

“既然是最后一程，那便走吧。夜莺酒吧还在等您，我们不该迟到。”

# 38. 暮光与晨曦

伴随着V12引擎低沉的轰鸣声响起，雨燕发动启动了这一头苏醒的黑豹，车灯劈开雨幕，平稳地滑入了大桥的主干道。

车厢内死寂一片，只有仪表盘发出的幽蓝光芒映照着雨燕冷峻的侧脸。

红夫人扯出一个惨淡的笑容，声音颤抖却带着惯有的傲慢，她终于给出来迟到的回复，在许久之后。

“一切似乎比预期……慢一点。或许至少……还有时间……还可以谈个交易。”

她的目光扫过雨燕完好无损的黑色皮衣，尽管身底的寒意在一瞬间已然炸开，直冲心脏，但那也将所有的痛觉都在一瞬间冻结。

“交易？你还有什么可交易的？你的命已经是我的了。”

雨燕诧异的看着眼前的女人，她的要求似乎有点多了，但后续她的话语还是让她发生了些许改观。

“我需要……一件衣服。一件能让我……体面地走完最后一程……的衣服。湿身……去……太失礼了。” 

雨燕沉默片刻，透过后视镜种飞驰的都市夜色，审视着眼前这个曾经高高在上、如今却在死亡边缘挣扎的女人。

她在思考，在权衡利弊，并不是来源于眼前的这个人的威胁，而是在思考。

红夫人她需要去补给，那是对她计划的一种干扰，但优雅存亡，关乎她的生死抉择，又至关重要。

她的“体面”和“王国”，一个人最后的执念，似乎可以满足这样的需求。

“银翼广场，在那里换好，送你回夜莺。” 

这不是交易，只是一个赠品，但在那之前，你需要这个！

雨燕顿了顿，从扶手箱里拿出来一个小巧的玩意儿，以及一枚古朴的玉扳指，给虚弱中的红夫人递了过去。

这是她控制局面、确保计划顺利进行的保证，虽然不会改变她注定死亡的结局，但也不会让她死在半路，破坏了红夫人自己选择的“舞台”。

“在那之前，你需要这个。”
