# 01

紫宸328年，冰天雪地的昆仑山脉之中，那白雪皑皑的雪山之巅，狂风如刀，冰雪刺骨，在那个本应该天寒地冻寂寥无人的冰天雪地，此时此刻，尽人山人海。

这里本应该是人迹罕至的龙脉之巅，江湖在这里绝迹，故事在这里终结，但此刻，在那肆意翻涌的云海之上，在那暴风雪都未曾凌虐的世界屋脊之上，此时此刻却无一例外的聚集了数不尽的江湖侠客。

一切不为其他，只为那一场惊天动地的大决战 ，一个名为萧婉的天下第一的侠客和那异域魔头拓跋烈的约战。

他们的到来，让这里人迹罕至变成了人山人海，热闹非凡，即使这里仍旧狂风肆虐，风雪入刀，但大战的即将来临，让这亘古不变的荒原多了一分新的故事。

此时此刻，只见那天下第一的侠客，遗世而独立于那天山之巅的冰岩之上，而她的对手，却依旧始终在她对面那三十丈之外的荒脊之上。

她们两人周边近百丈，风似乎已经停止了流动，雪似乎都已经凝固在了空中，但那并不是似乎，只是两个江湖侠客大战将至的内力余波。

只观那峰顶之巅的萧氏，一身深如夜色的玄锦劲装，在这冰天雪地的苍穹之巅分外显眼，而她那袖口领口金丝银线的繁复流云滚边，似乎在说明她那不俗的江湖身份。

不仅仅是天下第一，又或者有着其他的那不为人知的独特身份，而这，已经不是这观战之人所能知晓的秘莘了。

而这真正的秘莘却是，她那一身天下仅此一套的暗夜流光，是出自苏杭织造局最顶尖绣娘之手的孤品，仅为皇室专供，绝无仅有。

而在这天山之巅的漫天惨白中，她就像是一滴浓得化不开的墨，孤傲，决绝。


# 02

在她的对面三十丈之外，眼前正是那名名震西域，威震八方的异域魔头-拓跋烈，一个自称同样自称刀法天下无敌东方不败的第一人，只不过他的刀，被武林正派称作“魔刀”。

在他们一帮武林正派的眼中，对面的拓跋烈口中喘着粗气，大口大口的喷吐着异域的寒霜，他那浑浊的双眼似乎透漏着力竭的征兆，似乎胜利的天平已经向着这边倒下，而他的招式似乎早已被那萧婉所完全破解，一招一式之间，似乎尽是应对之策。

但殊不知，困兽犹斗，他那浑浊的双眼，同样也掩映着嗜血的光芒，只是不止，眼前的这个看似娇滴滴的武林弱女子，竟然可以在她那狂风暴雨一般的刀光之下制程仅百招。

“红尘客，你的剑很快，但你的气力想必也快尽了吧！”

眼前的拓跋烈狞笑着，身前满脸的横肉随着笑声肆意颤抖，手中的九环大刀随之发出刺耳的撞击声。

萧婉没有说话，正如眼前的刀客所言，经此一役，她的气力的确也消耗大半，虽然没有到那力竭的地步，但依旧和其所料不差！但她没有露怯，只是微微侧首，伴随着髻上的白玉流苏轻轻的晃动，寒风之中的那一阵悦耳的脆响。

此时此刻，她的眼神平静如水，神色之中却透漏着那一种居高临下的悲悯，让他的敌人无法摸清她的底牌，但那同样也是她那刻在记忆深处的对决方式。

在她的印象之中，她的战斗，向来都没有输过，她的内力，天下第一，她的武功，唯我独尊，她的力气，深不可测，她不会输，所以她居高临下，那是刻在骨子里的傲慢，既是出于对自己的自信，又是那源于本家的皇室底气。

但此时，不得不说，她已身处险境，眼前的对手，不，那是悍匪，能力已经超出了她的想象。

但即便如此，她依旧视其为蝼蚁！

“尽没尽，试过便知。” 

她淡淡开口，声音清冷如碎玉。风骤起，一瞬间，拓跋烈动了，而她也动了。

拓跋烈的人与刀合二为一，化作一道黑色的飓风,风卷残云，吹尽万物，毁天灭地，卷起千堆雪，淹没了眼前的她。

“天魔解体~”

观战之人这才后知后觉的报出了拓跋烈的拼命大招，发现了他那出其不意的巅峰武学，那融合他那毕生功力的巅峰一击。

萧婉也没退，意料之中，她迎着那道飓风冲了上去，或者换个方式，拿到飓风盯上了她，她也逃脱不掉！

手中的“流光”软剑早已出鞘，应对这魔刀客那愈发凌厉的攻势，柔弱的软剑在内力的加持下变得笔直，但却无法应对那愈发凌厉的刀锋，霎那间，刀剑相交，两人交错，但同时，天地间风云突变，时间彷佛瞬间静止。

山巅的积雪被瞬间清空，凌冽的冰岩瞬间粉碎，喷涌的剑气清空了眼前的一切，但唯独少了那惊天动地的爆炸声。

寒风很快再度占据了山巅，伴随着那一声极其轻微的“噗嗤”声，两人背对而立。

“谁赢了？”

风雪悄然退却，众人清楚那声音是利刃入肉的闷响，但由于他们的速度实在太快，却不知眼前的战局结果。


一瞬间天地风云突变，却不知眼下站局结果几何？众人惊讶的看着百米开外收刀负手而立的两位侠客，不知所言。他们外溢的真气让这一切细节已然成为了一场迷局，而真正的胜负却在毫厘之间已然尘埃落定。


风雪骤停，时间仿佛在那一刻静止，但那声利刃入肉的闷响却在悄然之中注定了一方的胜局，伴随着拓跋烈手中的大刀逐步颤抖，悄然落地，站局的落幕终将拉开序幕...

# 03 


赢了吗？所有的人似乎都以为是这个结局，但眼下只有她自己方才知道，输了。 

方才的战斗招式在那一瞬间千变万化，转折也在那电光火石的一瞬间，虽然战局的最后自己是险胜了那对面一招，将对面之敌斩首马下，但依旧还是落了半招防御之殇。

拓跋烈的刀气还是太过霸道，冲破了她的防御。她也仅仅是落了半招，避开了刀锋直接入体的惨烈下场，但那长虹贯日一般的凌厉刀气还是冲破了她的护身罡气。

那一股至阳至刚的刀锋所在的锐气，已经无声无息之间贯穿了她的左胸，击碎了她的傲气的同时，也残酷而精准的切断了她的心脉，让她的心脏无声无息之间既可化身为两半。

即便如此，眼下的她和他都没有倒下，一方面是大战之后身体内那汹涌澎湃的狂暴真气，另一方面是那场酣畅淋漓的大战之后战至终章的激素狂流，即使脑袋被斩首，又或者心脏被洞穿，但依旧横刀立马。

但一切终究是要结束，伴随着拓跋烈颤抖的手再也握不住那大刀，长刀哐当一声落地还是宣布了这场战局的结束，至少是那表面上的结束。

咽喉之上开始出现那细细的红线，伴随着他那双手捂着咽喉的动作，在那么一瞬间，观战的众人就已经明析了这场战斗的结局。

伴随着说书人宣判战局的结果，眼下的拓跋烈再也没有了之前负手而立的淡定。眼下的他，绝望的用手捂着咽喉，用全身的真气护住那澎湃的血流，但依旧无法抑制咽喉那红线的扩散。咽喉那红线扩散的长度像是那一道艳丽的胭脂痕，逐步爬上他的肩头。没有喷溅出来的大量的血迹，只是因为他那竭尽全力护体谋生的生存欲望，但喉咙里发出“咯咯”的异样以及轰然倒下的身躯，只能说这一切都只是一场枉然。

但此时生前对面另一边的对手萧氏，同样也面临着同样的问题。即使在别人看来，她依旧是保持着出剑后的姿势，长剑斜指地面，目光睥睨一切，长剑如风，剑尖不沾一丝血污，但身体内后知后觉传来的反馈依旧让她觉察到了那一切。

胸口处那透彻心扉的一缕寒意，伴随着那扎心一般的剧烈疼痛，眼下她的心脏彷佛被那敌人的双手所用力撕扯，每一次心脏的跳动，都在刺激着她那敏感的身躯。

“唔……” 她的脸色苍白如纸，身形也是微不可查的晃动了一下，但很快，她便意识到了这一切，和那拓跋烈一样，抽调那护体的真气，抑制住了身体的异样。

眼下的她，身体似乎也要立刻倒下，和那已经软倒在地的拓跋烈一样，因为这是身体里与生俱来的求生本能，毕竟或许只有这样，才可以减少真气的消耗,谋得那并不存在的一线生机。

但她那源于骨血里的骄傲不允许，因为只有她自己知道，她是紫宸王朝的长公主，骨子里是那天生的天潢贵胄，不能，也不应该轻易的倒下。

# 04 

尽管必须立刻倒下，这是身体求生的本能，但她的骄傲不允，她即便是死，也不能想眼下的那个草莽一样倒在这个冰天雪地的天山之巅，让那冰雪将自己无情封禁。

“起。”

她在心中默念，身体内的护身罡气被自己下意思的主动调用，磅礴的真气如同那一层坚不可摧的城墙，将那身体的异样强行遮掩，消弭了那心头的剧痛，维持了心脏的跳动。

和那天山之巅的拓跋烈一样，她同样也下意识的使用那身体内那股从小修炼、最为精纯浩瀚的“紫宸真气”来维持生命的延续，只不过，这次她做到了。

涌出的鲜血被那磅礴的内力所封禁，心脏上那破碎的伤口也被强行粘合，心口上那股透骨的寒意被她用绝强的意志力压了下去。而脸颊上那一抹红晕重新浮现开来，尽管那是她回光返照的艳丽之舞，显得那么不自然，但这至少是她续命的枷锁。

她缓缓直起身，动作轻微的像是如履薄冰，但依旧优雅地挽了一个剑花，身后的“流光”入鞘，她抬手理了理鬓角的碎发，动作从容得仿佛刚刚只是折了一枝梅花。

# 05

“红尘姑娘！”

“盟主！”

远处，几道急促的呼喊声很快就打破了大战之后的沉寂。风停雪落之后，远方观战的七八道人影施展轻功，飞掠而来，确认这场战局的落幕。

他们正是此次随她一同围剿魔教的江湖豪杰，也是为数不多的她的江湖挚友，其中有使得一手好快剑的江南少侠宋子安，也有豪爽粗犷的漠北刀客铁勒。

侠士落地，首先映入眼帘的是那已然倒地气绝的拓跋烈，他们无一不露出惊骇与狂喜的神色。

“这老魔头……真的死了？”宋子安上前探了探鼻息，随后兴奋地抬头，“红尘姑娘，你做到了！这一战，足以让你位列琅琊高手榜首！”

众人纷纷围了上来，眼中满是崇拜与敬畏。他们崇拜她着天下第一的名号又一次得到了落实，又同样敬畏她那不俗的身手和那未知的背景。

而那萧婉只是站在人群中央，看似在享受着这属于“大侠”的荣光，她嘴角含笑，目光温和，但殊不知她也仅仅是在拼命压抑身体内那逐渐散发的死气。

众人上前，她却不着痕迹地后退了半步，拉开了一点距离。她怕他们发觉他身上那浓烈的血腥气，但同样也秉持住了那一分天下第一固有的矜持。


“不过是侥幸罢了。”萧婉轻描淡写地说道，她没有解释太多，成王败寇，拓跋老贼的尸身就在这边，她也懒得去纠正殊胜谁败的结论，只是竭尽全力维系那平稳的声音，让众人听不出一丝虚弱，“眼下这老魔刀法虽猛，却失了灵动。我也只是胜在兵器之利罢了。”

“姑娘太谦虚了！”铁勒大笑，声如洪钟，“这可是‘魔刀’啊！咱们为了追杀他，在这昆仑山冻了三天三夜。如今大功告成，怎能不痛饮一番？”

说着，他解下腰间巨大的酒囊，晃了晃，里面传出哗啦啦的水声。

“这是我从漠北带来的‘烈火烧’，最是驱寒！今日红尘姑娘立此大功，必须满饮此囊！”

酒？

萧婉的睫毛微微颤动了一下。

她在评估眼下她的情况，此刻她的心脉全靠真气吊着，哪怕是一口热茶，都可能冲散真气，导致伤口崩裂。那烈酒入喉，无异于饮鸩止渴，大概率冲散那护体的真气，加速她的死亡，让她那为数不多的寿命在打折扣。

身体内的理智告诉她，该拒绝，应该立刻离开，但下一秒，她有释然，离开走向何方，找个地方疗伤嘛？还是默默的等死。

但同样，胸口那刺骨的凉意也告诉她，她需要那一口烈酒。不仅仅是周边挚友那一双双真挚、热切、充满仰慕的眼睛，闪现出来的那在冰冷的皇宫中从未见过的光芒，也是她出于对那身体感性的评估。

在那冰冷的紫宸皇城之内，众人敬她、怕她、算计她，似乎并没有将她视作生死之交，但下一刻，她也想通释然了，此时此地此刻，这将是她江湖梦的终点。

既然是终点，何不让这场梦，做得再圆满一些？

萧婉眼中的犹豫一闪而逝，眼下的她，同样也需要那一口烈酒来驱散胸口的寒意。想到了这些，取而代之的便是她那一抹极其绚烂的笑意。冰雪之巅，她那突如其来的明艳的笑容得让这漫天飞雪都失去了颜色。

“好。”

她伸出纤纤玉手，接过那只粗糙的皮酒囊。那只手白皙修长，与酒囊形成了鲜明的对比。

“今日大捷，当浮一大白。”

众人诧异，他们见识过红尘侠客天下第一的功夫，但见到她那冰山美人一般的笑容确实首次。


# 06 

众人很快寻得了一处避风的岩洞。铁勒手脚麻利地生起了篝火，枯枝在火中噼啪作响，驱散了几分山巅的寒意。

酒囊在众人手中传递，每一个人都喝得面红耳赤，高谈阔论。

“红尘姑娘，等回了中原，我一定要请你去我家乡的醉仙楼，尝尝那里的桂花鸭！”宋子安面若桃花，眼神晶亮地看着萧婉，“到时候，咱们再切磋几招剑法，我想向你请教那一招‘凤点头’。”

萧婉并没有做太多，只是默默的坐在火堆旁，伴随着眼底的笑意，火光映照着她绝美的容颜，为她苍白的肌肤镀上了一层虚假的暖色。

她轻轻的把持着手中的那酒囊，思索着眼下的下一步归途，但下一刻，她那轻微用力而微微泛白的双手悄然灵动。

“好啊。”她笑着应道，声音温柔得像是一场易碎的梦，“若有机会，定去叨扰。”

若有机会……她在心里默默补了一句：可惜，再无机会了。

轮到她了。

所有人的目光都聚焦在她身上，那是对英雄的期待。

萧婉举起酒囊，仰头。

那辛辣滚烫的酒液随即入口，汹涌的热力瞬间化作一道火线，顺着喉咙烧遍她的全身，很快便驱散了她那胸口的凉意。

“好酒。”

她放下酒囊，用衣袖优雅地拭去嘴角的酒渍，露出了一副极其享受的神情。那动作，既有江湖儿女的豪迈，又带着一种浑然天成的贵气，看得众人有些痴了。

一口烈酒下肚，她身体内的暖意似乎又是回归了几许，她似乎感觉状态又好了几分...

“红尘姑娘，你这身功夫究竟是师承何处？”有人借着酒劲问道，“江湖上都传你是隐世宗门的弟子，怎么从未听你提起过师门？”

萧婉摩挲着酒囊粗糙的表面，眼神变得有些飘忽。

师门？

她的师父是紫宸皇宫里那位只手遮天的老太傅，她的练武场是铺满金砖的大内演武厅。她的一招一式，来源皆是用来守护那个腐朽王朝的不传之密，而不是用来在这江湖里快意恩仇的江湖把式。

“我的师门……”萧婉轻笑一声，眼中闪过一丝不易察觉的落寞，“在一个很高、很远的地方。那里全是金色的笼子，住着一群飞不出去的鸟。”

众人听得云里雾里，只当她是喝醉了说胡话，纷纷大笑。

“姑娘真会说笑！若是笼中鸟，哪有姑娘这般翱翔九天的风采？”

“就是！姑娘是天上的凤凰，哪有笼子关得住你？”

凤凰……

萧婉心中一颤。

是啊，她是紫宸的凤。凤凰涅槃，浴火重生。可她这只凤，燃尽了羽翼，却再无重生的可能。

酒过三巡。

萧婉感觉到自己的视线开始有些模糊，酒精的热力开始体现，护体的真气开始紊乱。而第一步，则是从她那开始发麻的指尖开始。

她知道，这是身体快要到达极限的时刻，她再待下去，她就会控制不住体内翻涌的气血，会在这些人面前吐血倒地，露出那个血肉模糊的伤口。

她不能让他们看到那个画面，那个画面太血腥，也太残忍，不符合她“红尘客”的天下第一毫无败绩的身份，也不符合她紫宸长公主的体面，所以她必须在哪最完美的时刻退场。
