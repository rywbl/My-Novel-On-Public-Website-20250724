# 雪烬
夜幕如墨，覆盖着这座被遗忘的都市。霓虹灯在雨雾中破碎成斑驳的光点，像垂死星辰的残影。街道上，腐烂的躯体拖着断肢爬行，低阶异变者发出嘶哑的吼叫，而黑帮的武装巡逻车在废墟间穿梭，枪声如同丧钟般敲响。
在这片混沌之中，她站在一栋废弃高楼的顶层阳台，背对城市，面朝深渊。
她的名字早已被人遗忘。人们只称她为“白羽”——因为每当她在黑暗中现身，总披着那件厚重的白色羽绒服，帽檐边缘一圈蓬松的银白狐毛在寒风中轻轻飘动，宛如天使降临人间。
可谁又知道，那洁白之下，正酝酿着一场无声的崩坏？

# 第一章：血色暖绒

灰烬城的冬夜，从不温柔。
寒风像刀子一样刮过废弃的广告牌，金属支架发出呻吟般的吱呀声。街角的自动贩卖机早已锈蚀，玻璃碎裂，只剩下一具空壳，里面卡着半瓶发绿的汽水，瓶身爬满霉斑。远处，警报声断断续续地响着，像是这座城市垂死的心跳。
午夜零时，灰烬城最深处的地下,“猩红回廊”俱乐部正沸腾。
这里曾是战前一座废弃歌剧院，如今被铁爪改造成末世最后的奢靡殿堂。水晶吊灯悬在穹顶，虽已半数碎裂，残存的灯泡仍投下迷离紫光；舞池地面铺着从富豪别墅拆来的大理石，缝隙间渗着干涸的血迹与香槟残液。电子乐混着低吼与尖叫，在空气中震颤，仿佛整座建筑都在随着节拍腐烂。
而在那座歌剧院西北角一栋被藤蔓缠绕的旧公寓楼顶层，一盏孤灯亮着。
窗帘拉得严严实实，只透出一丝微弱的暖黄光晕。
她坐在镜子前。
镜中的女人有着一张足以令任何男人失神的脸：瓷白肌肤，高颧骨，唇形如弓，眼尾微微上挑，睫毛浓密得像鸦羽。可就在那完美的左脸颊下方，一道细如蛛丝的裂痕正悄然蔓延——皮肤微微隆起，边缘泛着不自然的青灰色，轻轻一碰，便渗出淡黄的组织液。
她迅速用棉签蘸取酒精擦拭，动作轻柔却果断，仿佛那不是自己的脸，而是一件需要修复的瓷器。
“第七天。”她低声说，声音沙哑却带着奇异的韵律感，“神经传导速度下降12%，瞳孔对光反应延迟0.3秒……肌腱开始钙化。”
她翻开摊在桌上的笔记本，纸页泛黄，字迹工整得近乎偏执。每一页都记录着身体的变化：体温、心率、皮肤溃烂面积、幻觉频率……甚至包括“今日是否梦见母亲”。
今天没有梦。
她合上本子，起身走向衣架。
衣架上挂着那件厚重的白色羽绒服。
它看起来崭新如初，尽管已经穿了三年。外层是防风防水的高密度尼龙，内里填充的是90%白鸭绒，领口一圈银狐毛柔软蓬松，在灯光下泛着珍珠般的光泽。这是父亲在她十六岁生日那天送的礼物，那时世界还未崩塌，他还在米兰做高定设计师，而她还在学小提琴。
“‘穿上它，就像披着云走路。’”
她模仿着父亲温柔的语调，嘴角微微上扬，随即又僵住——笑会牵动面部肌肉，而肌肉正在坏死。
她先穿上那件黑色礼服。
礼服是无肩带设计，深V领，后背开至腰窝，缎面如夜色流淌。这是她最后一次参加上流晚宴时穿的——那场宴会之后三天，第一例“X-7病毒”感染者在市中心地铁站暴走，撕咬了十七人。如今，这件礼服成了她的“伪装服”。它足够奢华，足够性感，足以让任何男人在靠近她时放松警惕；也足够紧致，能压住皮肤下偶尔鼓起的脓包，掩盖那些不该存在的肿胀。
她系好背后的暗扣，再套上羽绒服。拉链一拉到底，帽子戴上，整个人瞬间被包裹进一片柔软的白色之中。寒意被隔绝在外，连体内的刺痛似乎也暂时沉寂。
她走到窗边，掀开一角窗帘。
楼下街道上，两个低阶丧尸正围着一具尸体撕扯。它们的动作机械而贪婪，腐肉从指缝间滴落。
其中一个穿着破烂的羽绒马甲，另一个裹着超市塑料袋——滑稽又悲凉。
“原来我们都怕冷。”她喃喃道，“连怪物也想保暖。”
忽然，手腕上的通讯器震动起来。
她的眼神骤然锐利。
她转身走向武器柜，取出那把改装过的Mk18突击步枪。枪托上刻着一行小字：“For J.”——那是她妹妹的名字。妹妹没能撑过感染初期，死在她怀里，临终前说：“姐，别变成它们。”
她给枪上膛，检查弹匣，将备用弹夹塞进羽绒服内侧的暗袋。然后，她从抽屉里拿出一支口红——正红色，哑光质地。对着镜子，缓缓涂上。
鲜艳的红，与苍白的肤色形成惊心动魄的对比。
她最后看了一眼镜中的自己：天使的面容，恶魔的宿命，裹在一件温暖的白色茧中。
“今晚，”她轻声说，“我要拿回属于我的东西。”
推开门，寒风扑面而来。
她的身影没入夜色，羽绒服在风中鼓动，像一片不肯坠落的雪。

# 第二章：夜宴如刃

她，站在舞池边缘阴影里。
一袭黑色礼服紧裹身躯，无肩带设计勾勒出锁骨如刃，深V领口几乎开至心口，裙摆高开衩，露出修长双腿，脚踩一双镶钻高跟鞋，鞋跟中空，藏有两枚毒针。
但最引人注目的，是那件披在礼服外的白色羽绒服。
它不合时宜，却又奇异地融入这场荒诞盛宴。蓬松的银狐毛领衬得她脖颈愈发苍白，羽绒服未拉拉链，任由内里的黑色缎面礼服在霓虹下泛光。有人以为她是来走秀的疯子，有人猜测她是某位大佬的新宠——没人想到，这身华服之下，皮肤正以肉眼不可见的速度溃烂，神经末梢传来阵阵冰针般的刺痛。
她的右手垂在身侧，看似随意地搭在羽绒服口袋边缘。
可那口袋里，藏着一把折叠式突击步枪——枪身漆黑，消音器藏在袖口改装的金属管中，扳机连着她小指上的戒指。只需一个微小手势，子弹便能穿透三个人的颅骨而不惊动舞曲。
“你迟到了。”
一个沙哑的声音从背后传来。
她没回头。镜面墙映出对方身影，疤喉，铁爪帮副首领。
“堵车。”她轻笑，声音像融化的黑巧克力，“路上有几只低阶货挡道，顺手清了。”
疤喉眯起义眼：“你看起来……不太对劲。脸色太白。”
“今晚妆浓。”她抬手，指尖掠过脸颊——动作优雅，实则在确认那道新裂痕是否被粉底遮住。
音乐骤然切换，一首慢节奏的合成器情歌响起。人群自动分开，形成一条通道，直通中央舞池。
“陪我跳支舞？”
好啊。”她将左手搭上他的手背，右手仍插在羽绒服口袋里，手指已扣上扳机环。
两人步入舞池。
灯光转暗，只剩一束追光打在他们身上。疤喉的手搭在她腰间，力道带着威胁；她的身体贴着他，却如一片羽毛般轻盈疏离。每一步旋转，她都在计算距离、角度、逃生路线。
“你知道吗？”疤喉凑近她耳畔，呼吸带着腐臭，“上个月有个女猎人，和你一样漂亮，也穿黑裙子。
可结果呢？半夜在床上变成丧尸，咬断了雇主的喉咙。”
“真可惜。”她微笑，“我向来睡得很浅。”
下一秒，警报炸响！
不是俱乐部的警报，而是她体内的生物警报——病毒活性飙升！手臂肌肉开始痉挛，指尖发麻。
她必须速战速决。
“其实，”她忽然踮脚，在疤喉耳边轻语，“我今晚来，不是为了跳舞。”
话音未落，她猛地后撤，右手从羽绒服中抽出步枪，枪托狠狠砸向疤喉咽喉。同时，高跟鞋猛的弹出毒针，刺入他小腿。
你感染了！你是感染者！”
舞池瞬间混乱。保镖冲来，宾客尖叫逃窜。
她却站在原地，缓缓拉上羽绒服的拉链，将黑色礼服与染血的肌肤一同封存。帽檐拉低，银狐毛遮住半张脸。
“是啊，”她举起步枪，瞄准疤喉眉心，声音冷静如冰，“但我还没变成怪物，而你们，早就不是人了。”
枪声淹没在音乐中。
第一颗子弹正中眼眶，第二颗贯穿心脏，第三颗射向通风管道，硝烟弥漫，舞池化作修罗场。
她转身离去，白色羽绒服在血泊与碎玻璃中拖曳出一道洁净的轨迹。身后，有人喊她名字。
她没有回头。
在这座以欲望为燃料的城市里，天使与丧尸，往往只差一件衣服的距离。
而她，选择用羽绒裹住腐烂，用礼服伪装杀意，用枪声代替哀悼。
因为今夜之后，她或许真的会变成它们。
但在那之前,她仍是猎人。

# 第三章：夜之猎手

她把黑色礼服穿在内层，丝滑的缎面贴合肌肤，勾勒出完美的腰线与肩颈曲线。外搭羽绒服，既防寒，也遮掩。
当她行走于街头，没有人会想到，这身华美装扮之下，藏着一个正在腐烂的身体。
“你看起来不像个感染者。”曾有黑帮头目这样评价她。
“因为我不是。”她微笑，红唇微启，眼神却冷如冰霜，“我只是……比别人更懂得如何活着。”
她的话音在空旷的街道上消散，如同她呼出的白汽。黑帮头目死了，死在她的微笑和冰冷的枪口下。但她没有感到丝毫胜利的喜悦，只有深入骨髓的疲惫与寒冷。
她走出“猩红回廊”时，雨刚停。
街道空寂如墓穴。霓虹残影在湿漉漉的地面上扭曲晃动，映不出她的倒影。
因为她始终走在阴影里，步伐轻得像一片雪落。
白色羽绒服依旧洁净如初。
银狐毛领在夜风中微微拂动，没有一丝血迹，没有半点污痕。
不是因为她避开了战斗，而是因为她从不允许任何肮脏之物沾染这件衣服，那是她与“人”的最后契约。
体内的病毒却如潮水般翻涌，指尖传来针扎般的麻木。
“再撑两小时。”她低声自语，声音压得极低，仿佛怕惊醒体内沉睡的怪物。
目的地是城东的“旧钟楼”——渡鸦的藏身处。那位神秘医生从不收钱，只换一样东西：感染者临终前自愿献出的脑脊液样本。他相信，唯有清醒赴死的灵魂，才能为活人留下解药的线索。
讽刺的是，她现在正清醒地走向自己的终点。
她拐进一条窄巷。高跟鞋踏过碎玻璃，发出细碎脆响。忽然，头顶传来嘶哑的低吼。
三只低阶异变者倒挂在废弃电线杆上，涎水滴落，在积水里泛起油光。它们嗅到了她血液中那股熟悉的腐臭——同类的气息，混合着垂死的甜腥。
但她没有拔枪。
枪声会引来更多敌人，而羽绒服不能沾上硝烟之外的任何痕迹。
她轻轻解开羽绒服最上方的两颗暗扣，右手滑入内衬——那里缝着一个隐藏枪套，贴着黑色礼服的腰线。她抽出一把消音手枪，枪身漆黑如墨，握把刻着妹妹的名字缩写：“R.Y.”。
第一枪，无声。丧尸眉心多了一个小孔，倒栽而下。
第二枪，穿喉。第三枪，爆颅。
尸体落地时，她已侧身避开，白色羽绒服的下摆甚至未扬起半寸尘埃。她站在原地，静静看着脓血在脚边缓缓漫开，却始终未越雷池一步。
“你们已经死了，”她轻声说，“而我……还在选择怎么活。”
她重新系好羽绒服的扣子，将武器藏回内衬。整件外套依旧蓬松、洁白、一尘不染，仿佛刚才的杀戮只是幻觉。
路过一座坍塌的儿童游乐场时，她脚步微顿。
秋千在风中轻晃，一只破旧泰迪熊卡在滑梯缝隙里，纽扣眼睛掉了半颗。
那是她和妹妹常去的地方。
记忆闪回：阳光、棉花糖、小提琴声……还有地铁站外，妹妹临终前攥住她手腕的力道：“姐，别让我变成那样。”
她闭了闭眼，再睁开时，眸中只剩冰霜。
但羽绒服的领口，仍柔软如云。

# 第四章：雪下余温

旧钟楼矗立在灰烬河畔，塔尖倾斜，大钟停在11:59——末日降临前的最后一分钟。
她挣扎回到那间被藤蔓缠绕的旧钟楼，身躯下病毒侵袭已无法逆转。
推开铁门，门轴呻吟如叹息。关上门，隔绝了外界的喧嚣与死亡气息，也隔绝了自己作为“白羽”的身份。
她脱下羽绒服，小心翼翼地挂回衣架，仿佛那不是一件衣服，而是一个神圣的信物。然后，她解开黑色礼服的暗扣，任由它滑落在地，如一摊流动的夜色。
镜子再次映出她的身体。
这一次，她没有立刻处理伤口，而是静静地审视着。左颊的裂痕已经蔓延到了耳根，像一张精致的蛛网。腹部，原本被礼服紧紧束缚的地方，几块皮肤呈现出不祥的青紫色，下面有东西在微微蠕动。
“你比我预想的更糟。”渡鸦坐在轮椅上，白发如雪，“皮肤坏死率37%，神经传导中断，体温紊乱……你最多还有48小时。”
但她站得笔直，仿佛那具正在崩坏的身体与她无关。
窗外，月光穿过云隙，洒在那件叠放在椅背上的白色羽绒服上。银狐毛泛着珍珠般的光泽，洁净如初生之雪。
“给我针。”她终于说。
通讯器扫描了注射器上的微雕编码，一行小字浮现在空中。
“血清原型……K-03。”
这支，是她最后的希望。
“快注射！”渡鸦催促。
她没有丝毫犹豫，将针头刺入大腿静脉，针头缓缓推入液体。
起初，没有任何感觉。 一分钟后，剧痛如潮水般涌来！仿佛有无数只蚂蚁在啃噬她的骨骼，每一根神经都在尖叫。她咬紧牙关，指甲深深掐进手心，硬是一声没吭。身体剧烈地颤抖，痉挛，皮肤下的蠕动愈发剧烈，似乎要冲破束缚。
她蜷缩在地板上，汗水浸湿了头发，视野开始模糊。恍惚间，她仿佛回到了十六岁的那个冬天。
父亲带着她去阿尔卑斯山滑雪。她穿着他设计的那件白色羽绒服，在雪地里摔得鼻青脸肿。父亲没有扶她，只是站在不远处，笑着对她说：“Léa，记住，摔倒了要自己站起来。而且，要站得漂亮。”
他的声音温柔而坚定，像阳光下的积雪。
“你看，爸爸给你准备了什么？”他从怀里掏出一个保温杯，里面是热气腾腾的巧克力。“喝一口，就不冷了。”
那温热的液体滑入喉咙，驱散了寒意，也抚平了疼痛。
“爸爸……”她无意识地呢喃着，泪水混合着汗水滑落。
不知过了多久，剧痛缓缓消退，取而代之的是一种久违的温暖，仿佛泡在温泉里。她挣扎着爬起来，再次走向镜子。
奇迹发生了。
脸颊上的裂痕正在以肉眼可见的速度愈合、淡化，最终只留下一道浅浅的粉色印记。腹部的青紫褪去，皮肤恢复了光洁。她甚至能感觉到，坏死的肌肉纤维正在重新获得活力。
她成功了。她为自己又争取到了一点时间。
但她很清楚，这只是暂时的。K-03抑制剂的效果只能持续七十二小时。时间一过，病毒会以更凶猛的姿态反扑。
# 第五章：雪影之心
剧痛消退后，她躺在冰冷的石质地板上，胸膛剧烈起伏，贪婪地呼吸着钟楼里混合着尘埃与草药味的空气。渡鸦没有靠近，只是静静地在轮椅上看着她，眼中混杂着医者的审视与一丝不易察觉的的怜悯。
“感觉怎么样？”渡鸦的声音打破了沉寂。
她没有立刻回答，而是撑着地面，缓缓坐起。她伸出手，在月光下审视。皮肤光洁，指尖不再有那致命的麻木感，力量正重新回到肌肉纤维中。她站起身，走到那面落满灰尘的穿衣镜前。
镜中，那道从眼角蔓延至脸颊的裂痕已经消失无踪，只剩下一道几乎看不见的浅粉色印记，仿佛情人的抓痕。她抬手轻抚，皮肤下不再有蠕动的异物感，体温也恢复了正常。K-03血清的效果比她想象的更惊人。
“我感觉……像个活人。”她沙哑地说，声音里带着一丝恍惚。
“暂时的。这支原型血清的原理是‘超速再生’，它以透支你未来生命潜能为代价，换来72小时的‘假性痊愈’。时间一到，病毒会连本带利地反噬，你的崩坏速度会加快三倍。”
她对此并不意外，只是冷静回复：“走了~”
地下三层的武器库，墙壁上挂满了各式各样的枪械，从战前制式步枪到黑市改造的能量武器，一应俱全。
她没有选择那些威力巨大的重型武器。她径直走向最角落的武器台，那里放着一把她亲手改装的步枪。枪身基于Mk18的框架，但更纤细、更轻，枪管被缩短，加装了高效的内置消音器。最重要的是，它的每一个零件都被她拆解、打磨、重组过无数次，人与枪之间已经达到了完美的默契。枪托上，用小刀刻着她妹妹的名字缩写：“For J.”。
她为步枪换上特制的子弹——弹头由渡鸦提供，混合了水银、高浓度镇定剂和微量银粉，对异变者的神经系统有奇效。
她戴上黑色的战术皮手套，将备用弹匣塞入羽绒服内侧的暗袋，又挑选了几枚震撼弹和高能闪光弹。做完这一切，她从一个抽屉里拿出了那个泛黄的硬壳笔记本和一支笔。
她翻到新的一页，火光下，她的侧脸线条柔和，眼神却无比专注。 她写道：
“第十五天。K-03血清有效，身体机能恢复92%。倒计时71小时23分。新目标：***实验室，今夜之后，‘白羽’将暂时沉睡，代号‘雪影’，执行潜入任务。”
写到这里，她顿了顿，笔尖悬在纸上。火光跳动，映照着她绝美的脸庞，那道细小的裂痕仿佛又隐隐浮现。她知道，每一次任务都可能是最后一次。如果她死了，会有人记得她吗？记住这个在末世里，用腐烂之躯守护最后一点人性尊严的女人吗？
她深吸一口气，在页脚写下最后一行字：
“如果我死了，至少要让这个世界记住我的样子。”
她是“雪影”，灰烬城最顶尖的赏金猎人。 她的存在，本身就是一种证明。
她最后检查了一遍装备，将羽绒服的兜帽戴上。那张足以倾倒众生的脸庞被阴影覆盖，只有下颌的完美曲线和一抹倔强的唇色依稀可见。
推开武器库沉重的铁门，她沿着螺旋楼梯向上走去。钟楼的窗外，夜色正浓，暴雨将至。她的战场，在午夜之后。

# 第六章：钟楼之巅

子时，暴雨如注。
灰烬城被笼罩在一片水幕之中，霓虹灯的光芒被切割、揉碎，化作流动的光河。冰冷的雨水冲刷着城市的罪恶与血污，却洗不净空气中弥漫的腐朽气息。
她站在旧钟楼的顶层平台，任由狂风裹挟着雨点抽打在身上。那件白色的羽绒服外层是高密度防水尼龙，雨水落在上面，凝成水珠滑落，不留一丝痕渍。她像一座孤傲的雪雕，矗立在城市的最高点之一，俯瞰着脚下这座巨大的钢铁坟墓。
风在她耳边呼啸，仿佛无数亡魂的恸哭。她一动不动，与黑暗融为一体，耐心得像一个最老道的猎手。
一个黑影，毫无征兆地出现在她身后五米处。 
“你的胆子比我想象的还大，‘雪影’。”
她最后看了一眼远处的通天塔，那里是她此行的终点，或许也是生命的终点。但她的眼神里没有恐惧，只有燃烧的战意。
她拉紧羽绒服的拉链，转身，从钟楼的另一侧纵身跃下。
在坠落的瞬间，她扣动了手腕上的发射器，一道钢索呼啸而出，精准地钉入对面大楼的墙体。她的身体在半空中划出一道优美的白色弧线，像一只展开双翼的雪鸟，悄无声-息地滑向了城市更深的黑暗之中。
白色羽绒服在风雨中鼓动，像一片不肯坠落的雪。 今夜，猎人，已入场。

“纵使身躯腐朽，灵魂依旧滚烫。
我以羽绒裹尸，以礼服藏伤，以枪口指向黎明——
因为我相信，死亡不是终点，而是另一种开始。”
