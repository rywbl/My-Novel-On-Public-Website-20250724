
# 紫宸涅槃：开局假死后我重生了

 【重生/自我救赎/宫墙逆袭】
【女主绝境觉醒 + 模拟器读档重生 + 不必靠男人的大女主】
【不为复仇不为爱，只为学会爱自己。】
冰窖里，童姥死了，我也快了。
临死前一个“叮”，我得到了模拟器
无限回溯机会，方寸之地，可这熟悉的西夏为何变成了紫宸。
无崖子的画我烧了，
童姥的恨我放下了，
皇帝的江山我不要。
那些笑我丑的人？一杯茶教他们做人。
这辈子唯一要救的——
是我自己。
这一次，不为任何人活。
我是李秋水。
我的自由，我自己给。
【“她一生要强，到头来却发现自己爱错了人，也恨错了人。最可悲的不是得不到，而是一直以为得到了，其实从未拥有。”】
【真正的救赎不是让伤害过你的人道歉，不是让错过你的人回头，甚至不是毁掉你的仇人。而是在一堵墙内，你依然可以把自己活成全世界】
【不是让她“得到”无崖子的爱，而是让她明白——即使没有那个人的爱，她也值得被爱，也可以好好活着。】
