# 13 虚空之潮

瑶光通天塔的建设，成为了七绝圣域所有生灵共同的宏伟事业。塔基深植于地脉核心，每日吞吐着海量的灵石灵液；塔身则以肉眼可见的速度，一寸寸向着云霄攀升。每一个参与建设的生灵，都能清晰地感受到，随着塔的升高，圣域内的灵气运转愈发和谐，修炼时也愈发事半功倍。希望，如同塔尖的晨光，照亮了每一个人的心。

然而，就在通天塔建至三百丈，初具规模，塔身符文开始与七座主峰遥相呼应，形成一个巨大循环的那个夜晚，异变毫无征兆地降临了。

最先察觉到不对的，是青瑶。

她正在塔顶的临时石室中闭目调息，借助塔的力量梳理着整个圣域的灵气流向。突然，她的心神猛地一颤，仿佛一根无形的、冰冷的针，刺入了她与这片天地的紧密联系之中。

那不是妖气，不是魔气，也不是任何她所熟知的能量。那是一种……空洞、死寂、充满了混乱与无序的“无”。它就像一滴浓墨滴入清水，带着一种源自世界之外的、令人作呕的违和感，悄然渗透进了七绝圣域的边界。

青瑶猛然睁开双眼，幽绿色的眸子中神光暴涨。她一步踏出，身影已出现在塔顶的最高处，俯瞰着整个圣域。

夜色下的圣域，依旧被七色神光笼罩，看似平静祥和。但在她敏锐的感知中，圣域外围的黑暗，正变得前所未有的粘稠与沉重。平日里山林间正常的虫鸣兽吼，此刻竟已完全消失，取而代之的是一种令人心悸的死寂。

“传我谕令！”青瑶冰冷的声音瞬间传遍七大主峰的堂口，“战堂集结，阵堂启动所有外围警戒法阵，其余各堂进入最高戒备状态！有大敌将至！”

命令一下，整个圣域瞬间从修炼的狂热中惊醒，变成了一台高速运转的战争机器。战堂的妖修们迅速披上藤甲，拿起法器，在堂主的带领下奔赴圣域边界；阵堂的修士们则冲向各处阵眼，将一块块灵石填入其中，一道道半透明的光幕在圣域外围的山林间升腾而起。

就在这时，大地开始轻微地震动。

起初只是如同远方的闷雷，但很快，震动变得越来越剧烈，仿佛有千军万马正在奔腾而来。圣域边缘，那些刚刚亮起的警戒法阵，如同被投入石子的水面，开始剧烈地闪烁和扭曲。

“来了！”一名站在光幕后的狼妖瞳孔骤缩，发出了惊恐的吼声。

只见远方的黑暗林海中，亮起了一片片密密麻麻的、散发着猩红光芒的“星点”。那不是星星，而是眼睛！无数双充满了暴虐、饥渴与疯狂的眼睛！

“吼——！”

一声不属于此界任何生灵的嘶吼，撕裂了夜空。紧接着，无穷无尽的怪物，如同决堤的黑色潮水，从四面八方冲了出来。

这些，便是“界外异兽”。

它们形态各异，扭曲而恐怖。有的长着昆虫的节肢，却顶着一颗布满獠牙的鱼头；有的全身覆盖着流淌着黑色粘液的甲壳，背上长满了胡乱挥舞的触手；还有的干脆就是一团蠕动的血肉，上面随机地镶嵌着数十只眼睛和利爪。它们身上散发出的，正是那种让青瑶感到心悸的、属于“虚空”的混乱气息。

它们没有灵智，只有最原始的吞噬本能。它们的目标，就是七绝圣域这片散发着勃勃生机与浓郁灵气的“美食”！

“轰！轰！轰！”

兽潮狠狠地撞上了圣域外围的防御法阵。光幕剧烈地摇晃，发出不堪重负的嗡鸣。每一头异兽的撞击，都带着一股消融、瓦解灵气的诡异力量，让法阵的灵石消耗速度远超预估。

“稳住！阵堂的，加大灵石投入！”战堂堂主，一头修为高深的黑虎妖王怒吼着，他一爪拍出，巨大的妖力爪影越过光幕，将十几头异兽撕成碎片。但那些碎片在地上蠕动了几下，竟又重新融合成更小的怪物，继续疯狂地冲击着法阵。

“这些鬼东西杀不死！”有妖修惊恐地大叫。

青瑶立于通天塔之巅，将这一切尽收眼底。她的脸色无比凝重。她终于明白，那股未知的力量是什么了。那是“虚空”对“实体世界”的侵蚀！不知是何种原因，七绝圣域这片新生的、灵气过于璀璨的福地，成为了虚空侵蚀的突破口，吸引了这些无尽的虚空造物。

“咔嚓——”

终于，一道位于西侧的光幕，在数头巨型异兽的持续撞击下，迸裂出一道巨大的裂痕。

“守住缺口！”黑虎妖王咆哮着，率先冲了过去。

无数异兽如同闻到血腥味的鲨鱼，疯狂地从裂口涌入。一场惨烈至极的血肉搏杀，瞬间展开。

圣域的战士们虽然英勇，但他们从未见过如此诡异的敌人。他们的法术打在异兽身上，效果大打折扣；他们的利爪撕开对方的身体，却无法阻止其再生。而异兽的每一次攻击，都带着虚空的腐蚀之力，伤口极难愈合，甚至会不断侵蚀受伤者的妖力与生命力。

伤亡，在飞速扩大。

瑶光城内，无数生灵透过水镜法术看着边界的惨状，脸上写满了恐惧。他们刚刚拥有的家园，难道就要这样被毁灭吗？

“瑶主……我们的瑶主呢？”一个年幼的兔精颤抖着问她的母亲。

仿佛是回应她的祈祷，就在防线即将崩溃之际，一道通天彻地的七彩光柱，猛然从瑶光通天塔的塔顶冲天而起，贯穿了夜空中的灵气星云！

青瑶出手了。

她没有直接飞赴战场，而是盘膝坐在了塔顶的阵法核心之中。她将自己的神念，与整座初具规模的通天塔，与七绝圣域的地脉，与每一个正在运转《瑶光经》的生灵，彻底连接在了一起！

“圣域子民，借我尔等之力！”

她的声音，如同天道纶音，响彻在每一个生灵的脑海。

“愿为瑶主而战！愿为圣域而战！”

无需动员，发自内心的呐喊从圣域的每一个角落响起。无论是正在前线搏杀的战士，还是在后方瑟瑟发抖的妇孺，都在这一刻，毫无保留地运转起《瑶光经》，将自己的信念与力量，顺着那无形的连接，汇入通天塔中。

嗡——！

通天塔发出了震耳欲聋的轰鸣。塔身上，无数符文逐层亮起，仿佛一条苏醒的巨龙。七座主峰同时大放光明，赤、橙、黄、绿、青、蓝、紫，七道磅礴的灵力洪流冲天而起，在空中交汇，最终尽数灌入通天塔顶！

青瑶的白衣在狂暴的能量气流中猎猎作响，她的长发无风自动。她缓缓抬起手，对着那片被异兽肆虐的战场，轻轻一指。

“瑶光·净化！”

一道由纯粹的七彩灵力构成的巨大光环，以通天塔为中心，猛然扩散开来！

光环所过之处，时间仿佛都停滞了一瞬。那些狰狞恐怖的界外异兽，在接触到光环的刹那，身体如同被烈阳照射的冰雪，发出了凄厉的嘶鸣，从身体结构最基础的层面开始崩溃、消融，最终化作一缕缕黑色的虚空气息，被七彩灵光彻底净化、蒸发！

仅仅一击，涌入圣域的数万异兽，被清扫一空！

战场上，幸存的战士们目瞪口呆地看着这神迹般的一幕，随即爆发出震天的欢呼！

然而，青瑶的脸上却没有丝毫喜色。她清晰地感觉到，刚才那一击，几乎抽空了圣域法阵中三分之一的灵石储备，更是让无数修为低微的子民陷入了脱力状态。

而圣域之外，那黑色的兽潮，依旧无穷无尽，仿佛刚才的损失对它们而言，不过是九牛一毛。

“这样下去不行，”青瑶心中迅速盘算，“被动的防守，只会被活活耗死。必须找到这股兽潮的源头，那个沟通虚空的‘奇点’！”

她再次调动通天塔的力量，这一次，力量没有用来攻击，而是化作无形的感知，顺着那些虚空异兽来时的轨迹，向着黑暗的源头逆流探查而去。

很快，她在距离圣域三百里外的一处盆地中，找到了目标！

那里的空间，像一块破碎的玻璃，呈现出不规则的扭曲。一个直径超过百丈的、漆黑如墨的空洞悬浮在半空中，正如同一个巨大的疮口，源源不断地向外“呕吐”着界外异兽。空洞周围，盘踞着数头体型堪比山丘的巨型异兽，它们的气息，比其他异兽强大了百倍不止，显然是这股兽潮的头领。

“找到你了！”

青瑶眼中杀机一闪。她知道，必须斩首！

“黑虎、白鹭、猿通！”她向战堂的三位妖王发出了神念传音，“我将以通天塔之力，为你们打开一条通路，并暂时压制那片区域的虚空气息。你们的任务，是摧毁那个传送奇点！此战，关乎圣域存亡！”

“遵瑶主法旨！”三位妖王没有丝毫犹豫，发出了决绝的回应。

下一刻，青瑶深吸一口气，将双手按在了身下的阵法核心上。

“万法归一，神光为引，开道！”

通天塔顶端射出的七彩光柱猛然一转，不再是向外扩散，而是凝聚成一道无比凝实的能量光束，如同一柄创世之剑，撕裂了三百里的夜空，精准地轰击在那片盆地的上空！

光束并未直接攻击，而是在空中爆开，形成了一个巨大的七彩结界，将整个盆地笼罩其中。结界之内，虚空气息被强行压制、净化，那些普通的异兽行动变得迟缓，身体甚至开始出现不稳的迹象。

“就是现在！”

黑虎妖王、白鹭仙子、猿通长老，化作三道流光，沿着神光开辟的“轨道”，以不可思议的速度，瞬间跨越了三百里的距离，降临在盆地之中！

“杀！”

三位妖王毫无保留地爆发了自己最强的力量，向着那个漆黑的空洞冲去。

然而，守护在空洞周围的那几头巨型异兽，也从神光的压制中反应过来，发出了愤怒的咆哮，迎了上来。

一场惊天动地的大战，在盆地中爆发。

青瑶端坐塔顶，一心二用。她一边要维持着对盆地的神光压制，为三位妖王创造机会；另一边，还要调动圣域的防御法阵，抵御着依旧在疯狂冲击的兽潮。

这对她的心神与法力，是前所未有过的巨大消耗。通天塔的灵石储备，如同流水般消耗着。

盆地中，三位妖王陷入了苦战。那几头首领级的异兽，不仅力量强大，身体更是坚不可摧，甚至能小范围地扭曲空间，极难对付。黑虎妖王的一条臂膀被腐蚀得露出森森白骨，白鹭仙子的羽翼也变得残破不堪。

“瑶主！我来为他们创造机会！”猿通长老发出一声大吼，他那苍老的身躯猛然膨胀，现出了本体——一头顶天立地的通臂巨猿！

“法天象地！”

巨猿放弃了所有防御，用自己庞大的身躯，硬生生将三头首领异兽死死抱住！

“动手！”他用尽全身力气吼道。

“老猿！”黑虎妖王双目赤红，但他知道此刻不容犹豫。他与白鹭仙子对视一眼，将全身残存的妖力，汇聚于至强一击。

“虎王裂天杀！”

“白羽归墟剑！”

两道蕴含着毁灭气息的攻击，越过了巨猿的身体，狠狠地轰击在那个漆黑的空洞之上！

“轰——！！！”

无法形容的巨响传来，漆黑的空洞剧烈地扭曲、收缩，最终在一阵刺耳的尖啸声中，猛然向内坍塌，彻底消失。

随着奇点的毁灭，盆地内的所有异兽，以及正在冲击圣域的兽潮，都仿佛被抽走了灵魂，动作猛然一滞，随后身体开始不受控制地崩溃、瓦解，化作了漫天的黑灰。

危机，解除了。

但代价，是惨重的。

猿通长老的身躯在空洞坍塌的瞬间，被虚空之力绞得粉碎，形神俱灭。黑虎妖王和白鹭仙子也身受重创，气息奄一息。圣域边界，超过三成的战士阵亡，几乎人人带伤。而通天塔的灵石储备，消耗了近七成。

整个圣域，弥漫着一股悲壮的气息。

然而，就在此时，天地间，一股前所未有的浩瀚功德金光，从九天之上降下！

这是天地对此方世界生灵，成功抵抗虚空侵蚀的嘉奖！

庞大的功德金光，一分为三。最大的一股，如天河倒灌，涌入了通天塔顶的青瑶体内。她感觉自己那神魂深处的空洞，在这股温暖而神圣的力量下，竟被修复了大半！她的修为瓶颈，瞬间被冲破，一股更加深邃、更加强大的力量在她体内苏醒。

第二股金光，则笼罩了整个七绝圣域。所有在战斗中阵亡的生灵，其残魂被金光接引，获得了转生的机会；所有受伤的生灵，伤势瞬间痊愈，甚至修为都有所精进。

而第三股，也是最奇特的一股金光，化作了亿万光点，洒向了圣域内所有非人形的妖族与精怪。

在金光的沐浴下，那些灵智已开、修为达到一定程度的妖与精，身体开始发生剧烈的变化。他们的兽形或植物形态，在金光中溶解、重塑。骨骼在生长，血肉在衍化。

“啊！我的身体！”

“我……我长出了手！”

伴随着一阵阵惊喜的呼喊，无数妖族、精怪，褪去了自己原始的形态，化作了与人类无异的样貌。虽然有些还保留着些许特征，如头顶的耳朵、身后的尾巴，但他们，终于拥有了最适合修炼的道体——正式化形！

这突如其来的巨大惊喜，冲淡了战争的悲伤。无数新化形的生灵，跪倒在地，朝着瑶光通天塔的方向，发自内心地叩拜。

与此同时，七绝圣域的地脉深处，仿佛也受到了功德的感召，发出了剧烈的轰鸣。

下一刻，七道前所未有、粗壮无比的灵石洪流，从地底喷涌而出！这些灵石不再是普通的乳白色，而是呈现出淡淡的青色，其中蕴含的灵气，比之前的灵石精纯了十倍不止！

“中品灵石！”有识货的人类修士发出了不敢置信的惊呼。

这七道中品灵石洪流，如同七条青色的巨龙，盘旋着飞入通天塔，将那消耗一空的储备，瞬间填满，甚至犹有过之！

青瑶缓缓站起身，感受着体内前所未有的强大力量，看着下方那些获得了新生、跪地叩拜的子民，以及那源源不断生成的中品灵石。

她知道，经历过血与火的洗礼，七绝圣域，才算真正地站稳了脚跟。

一个更加辉煌，也必将面临更多挑战的全新时代，已经到来。
