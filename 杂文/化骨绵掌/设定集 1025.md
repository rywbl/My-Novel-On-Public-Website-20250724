# 主角：太后 

1. 参考鹿鼎记中的太后设定 ：雍容尊贵的“孝康皇后” 背景设置在紫宸王朝
2. 表面上她看上去只是个吃斋念佛、不问世事的深宫妇人，美女贵妇设定，对下属都颇为慈爱
3. 偶然被半夜潜入寝宫的有心之人(海大富)撞见练一种阴毒武功 化骨绵掌 


# 化骨绵掌 

1. 核心设定 “外似轻绵，内藏刚毒；中掌者当时浑若无事，两个时辰后骨骼寸断、脏腑碎裂，死状如烂泥。”
2. 它传自《天龙八部》星宿老怪丁春秋一脉，后被神龙教带到“蛇岛”，成为太后独门秘技
3. 发作机制：表面看掌风柔软、姿势舒展，实际把一股“阴柔腐蚀劲”打进对方体内；这股劲会顺着经脉慢慢渗透，像硫酸一样啃骨头，延迟两个时辰（约4小时）后集中爆发
4. 致死特征：受害者先是毫无痕迹，随后全身骨头“软如绵”，节节寸断，最后像一滩肉泥，无药可救；
5. 知道的人极少，练成的人更少： 杀人不见血、事后无证据 

# 样本

1. 四人死时“肌肤完好，无瘀无损，血脉如常”，但“骨骼尽酥，触之如绵”，御医只能写成“暴病卒”或“惊风卒”，根本找不到外伤或中毒迹象。
2. 多个中毒样本 孝康皇后、亲王、贵妃董鄂妃、贞妃等 皆为女性设定 接连暴毙
3. 使“化骨绵掌”，“中掌者两个时辰后骨节自解，脏腑碎裂而死”
4. 对着一名小太监隔空出掌，掌风“软绵绵的像抚灰尘”，小太监当时只觉胸口一热，仍好端端走开。 
5. “这人再过一个时辰就要骨软如绵，变成一滩人肉布袋。”——首次让读者直观感受到“延迟发作、外表无伤”的杀人特点

她的身躯在融化，她的骨肉在消融，在那华美的锦衣华服之下，她已然被那歹毒的皇太后用化骨绵掌所暗害，此时的她，仅仅是一副即将消融的华丽残躯，苟活于世


她的身躯在融化，她的骨肉在消融。在那华美的锦衣华服之下，曾经的她是紫宸王朝最受宠爱的贵妃，端庄而美丽，举手投足间尽显雍容华贵。然而此刻，她的脸色苍白如纸，双眼失去了往日的神采，唯有一抹不甘的泪痕滑落。

“为什么……”她的声音微弱，像是从喉咙深处挤出的最后一丝力气，“太后……为何如此狠毒？”

空气中弥漫着一股淡淡的药香，那是太后寝宫中常年燃烧的安神香。她的脑海中浮现出几日前的画面——太后那双慈爱的眼睛，轻轻抚过她的肩膀时的温柔笑意。谁能想到，那一掌之下，竟藏着致命的杀机。

“化骨绵掌……”她喃喃自语，终于明白了这四个字的真正含义。那股阴柔的掌劲，像一条无形的毒蛇，悄无声息地潜入她的经脉，缓缓啃噬着她的骨骼与脏腑。两个时辰前，她还在宫中散步，感受着秋日的微风；而现在，她的身体已经不再属于她自己。

她的手指颤抖着，试图抓住什么，却只触碰到冰冷的地砖。锦衣的袖口滑落，露出一截纤细的手臂，皮肤依旧光滑如玉，但骨骼却已然软化，像是失去了支撑的泥土。

“太后……”她的声音渐渐低沉，眼前的世界开始模糊。她知道，自己再也无法揭露那深宫之中的阴谋，再也无法为自己讨回公道。

远处，太后的寝宫灯火通明，隐约传来一阵悠扬的佛经声。那声音平静而祥和，仿佛什么都未曾发生。


夜风吹过，带走了她最后一丝体温。华丽的残躯静静地躺在地上，锦衣依旧光鲜，仿佛她仍是那个高高在上的贵妃。然而，生命的光辉早已熄灭，只剩下无尽的黑暗。
